registerWin()method 
public static function registerWin(win:DisplayObject, layoutType:int = 0, hOffset:int = 0, vOffset:int = 0):void

注册一个窗口布局（任意显示对象，窗口或面板或按钮控件） register a window layout(it can be any display object, windows or panels or button components)

Parameters

win:DisplayObject — 需要布局的显示对象 the display object you want to layout
 
layoutType:int (default = 0) — 布局类型，在舞台中心这样写：WinPosManager.V_CENTER | WinPosManager.H_CENTER layout type, use WinPosManager.V_CENTER | WinPosManager.H_CENTER for center of the stage.
 
hOffset:int (default = 0) — 水平偏移量 horizontal offset
 
vOffset:int (default = 0) — 垂直偏移量 vertical offset
