loadNow()method 
public function loadNow(url:String, format:String, callback:Function, callbackParams:Array = null, addCache:Boolean = false):void

立刻开启一个并发加载，如果URL在加载队列中则从队列中移除，如果format与队列加载中不一致，将使用本次调用的值

Parameters

url:String — 加载地址
 
format:String — 加载格式，URLLoaderDataFormat里的值
 
callback:Function — 回调函数，至少带一个Object类型的参数
 
callbackParams:Array (default = null) — 回调参数
 
addCache:Boolean (default = false) — 加入缓存，对于加载同一个URL多次调用，只要有一个调用使用加入缓存则会加入缓存
